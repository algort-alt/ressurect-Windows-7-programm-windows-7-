# ressurect-Windows-7-programm-windows-7-
откройте релиз "programm", скачайте архив 'ressurect.Win7.zip' и распакуйте его.

установив программу из этого проекта на систему Microsoft Windows 7 вы "воскресите" её. В 2020 году microsoft отключили поддержку windows 7,  и с тех пор множество программ не поддерживаются на windows 7 (например steam) а установив программу вы вернёте поддержку программ и бонусом установите браузер, который сможет загрузить любые сайты.

(смотреть код только по файлу python нет смысла, ведь он почти ничего не делает, этот проект является сборником нужных программ для поставленной цели, а программа из этого проекта просто устанавливает их, а затем попогает удобно их использовать.)
